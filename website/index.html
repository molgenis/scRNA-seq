scRNA-seq cis-eQTLs and co-expression QTLs

This web page accompanies the manuscript titled
'Single-cell RNA sequencing identifies cell type-specific cis-eQTLs and co-expression QTLs. ', by van der Wijst et al, published in Nature Genetics.
For further questions, contact the corresponding author: lude@ludesign.nl



Study

Single cell transcriptomics of PBMCs of 47 donors from the Lifelines Deep cohort (general population, Northern part of the Netherlands). Cells of five or six different donors were pooled together in one sample pool, resulting in eight different sample pools. In total, 28.855 cells were captured and their transcriptomes were sequenced to an average depth of 74k. Genotype data is available for each donor, which allowed us to use the Demuxlet method that uses variable SNPs between the pooled individuals to determine which cell belongs to which individual. Since genotype information is lacking of 2 individuals, the transcriptome of only 45 individuals could be retrieved.

Supplementary Data

Gene expression counts

Raw gene expression counts
Raw gene expression counts per lane in Matrix Market Exchange format. Each line of this file contains the index and value for every entry in the matrix, whereby the rows represent the genes (stored in genes.tsv) and the columns the cell barcodes (stored in barcodes.tsv). The Cell Ranger output for each lane has been included in the Cell Ranger web summary.

Demultiplexing

Sample identification
Cell barcodes for samples. This file can be used to link the barcodes to the correct sample. Each line contains a barcode. The first column contains the barcode, and the second column contains the sample identifier.

Cell type classification

Cell type classification
A two column tab separated file containing the cell barcodes (remained after QC) linked to their designated cell type.

eQTL analyses summary statistics

Raw output files of the eQTL analyses
The eQTL summary statistics for the eQTL analysis refined to previously reported top-eQTLs from whole blood deepSAGE (“restricted-eQTL-deepSAGE_PBMC”) and RNA-seq data (“restricted-eQTL-RNAseq_PBMC”) and the genome-wide eQTL analysis (“genome-wide-eQTL_cellType”). From left to right column: Nominal P-value, SNP name, SNP chromosome number and position, assessed allele, Ensembl gene name, HGNC gene symbol, gene chromosome number and position of gene center, Spearman’s rank correlation, Z-score, FDR for each gene-SNP pair that was tested (each analyzed cell type is shown in a separate file).

MAGIC imputation

Gene expression per donor after MAGIC imputation in the CD4+ T cells
Gene expression count matrices per donor after MAGIC imputation in the CD4+ T cells. Filenames indicate the sample IDs, columns represent the cells and rows the genes.

co-expression QTL summary statistics

co-expression QTL summary statistics
The co-expression QTL summary statistics of all results (significant and non-significant) for the eQTL analysis restricted to the 145 genes that showed a cis-eQTL effect in the CD4+ T cells. Interactions and correlations were only calculated for those genes that show variance in gene expression within the CD4+ T cells and are expressed in each of the 45 donors, i.e. 102 cis-eQTL genes were compared to 7,975 genes.

Data access

Gene expression data (EGA) and Genotype data (LifeLines Workspace)

Genotype and gene expression data are freely available. Processed (deanonimyzed) single-cell RNA-seq data, including a text file that links each cell barcode to its respective donor, has been deposited at the European Genome-phenome Archive (EGA), which is hosted by the EBI and the CRG, under accession number EGAS00001002560. Genotype data has been deposited in the LifeLines Workspace. The LLDEEP DAG2+ Data Access Committee (DAC) determines access permission. In order to request data access, please click on request data and fill in the data request form with a signed data access agreement (please note: signature of PI required).
